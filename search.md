# 🔍 Codeforces to Baekjoon 통합 검색 가이드 (v3.21)

본 확장 프로그램은 코드포스의 레이팅 시스템을 우리에게 익숙한 **백준(solved.ac) 티어 체계**로 변환하여 제공합니다. 이제 숫자가 아닌 티어로 문제를 검색하고 관리하세요.

---

## 1. 난이도 검색 (Tier Search)

가장 익숙한 티어 약어를 사용하여 문제를 검색할 수 있습니다.
- **약어:** `b`(Bronze), `s`(Silver), `g`(Gold), `p`(Platinum), `d`(Diamond), `r`(Ruby), `m`(Master)
- **등급:** 각 티어 뒤에 `1~5` 등급을 붙여 사용합니다. (숫자가 작을수록 높은 난이도)

| 문법 | 예시 | 설명 |
| :--- | :--- | :--- |
| **단일 티어** | `r:s3`, `tier:g1` | 실버 3 또는 골드 1 난이도 문제 검색 |
| **티어 범위** | `r:s5..g1` | 실버 5부터 골드 1 사이의 모든 문제 검색 |
| **이상/이하** | `r:p5..`, `r:..s1` | 플래티넘 5 이상 또는 실버 1 이하의 모든 문제 |

---

## 2. 해결 여부 및 상태 (@me)

본인의 제출 기록을 기반으로 문제를 필터링합니다.

| 문법 | 예시 | 설명 |
| :--- | :--- | :--- |
| **맞힌 문제** | `@me` | 내가 해결(OK)한 문제만 표시 |
| **안 푼 문제** | `~@me`, `!@me` | 내가 아직 해결하지 못한 문제만 표시 |
| **시도한 문제** | `?@me` | 제출 기록은 있으나 아직 해결하지 못한 문제 |

---

## 3. 정답자 수 및 태그

| 문법 | 예시 | 설명 |
| :--- | :--- | :--- |
| **태그** | `#dp`, `tag:math` | 특정 알고리즘 태그 검색 (공백은 `_` 사용) |
| **정답자 수** | `s:1000..` | 1,000명 이상이 맞힌 문제 검색 |
| **스마트 검색** | `greedy` | 제목이나 태그에 'greedy'가 포함된 모든 문제 검색 |

---

## 4. 논리 연산자

- **AND (공백):** `#dp r:s3` (DP이면서 실버 3인 문제)
- **OR (|):** `#math | #geometry` (수학 또는 기하학 문제)
- **NOT (~):** `~#dp` (DP 태그가 없는 문제)
