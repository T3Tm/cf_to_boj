# 📖 Codeforces 통합 검색 필터링 엔진 명세서 (solved.ac 스타일)

본 문서는 코드포스(Codeforces) 환경에서 백준(solved.ac) 스타일의 복합 검색을 수행하기 위한 검색 문법 및 논리 연산자 설계 명세입니다.

## 1. 기본 검색 문법 (검색어 조건)

### ① 문제 메타데이터 검색
* **`id:` (또는 문제 번호 직접 입력)**
  * **기능:** 특정 대회 번호 및 문제 인덱스로 정확한 문제 검색
  * **예시:** `id:4A`, `id:1520B`
* **`title:`**
  * **기능:** 문제 제목에 특정 문자열이 포함된 문제 검색 (대소문자 및 공백 무시)
  * **예시:** `title:watermelon`

### ② 알고리즘 및 난이도 검색
* **`tag:` 또는 `#`**
  * **기능:** 알고리즘 분류(태그)로 문제 필터링
  * **특징:** 한글 태그명 입력 시 내부 사전을 통해 영문 태그로 자동 치환 후 검색
  * **예시:** `#math`, `tag:수학`, `#dp`
* **`tier:` 또는 `*` (백준 스타일 난이도)**
  * **기능:** 브론즈~루비 형태의 친숙한 티어 시스템으로 문제 난이도 검색
  * **지원 범위:** 단일 티어, 범위(`..`, `-`), 이상/이하(`>=`, `<=`)
  * **예시:** `*s1..g5`, `tier:>=p5`
* **`rating:` 또는 `r:` (코드포스 순수 레이팅)**
  * **기능:** 코드포스 고유의 100점 단위 숫자로 난이도 검색
  * **예시:** `r:1500`, `rating:1500..2000`

### ③ 통계 및 유저 기반 검색
* **`s#` (해결한 사람 수)**
  * **기능:** 해당 문제를 맞힌 사람 수를 기준으로 필터링
  * **예시:** `s#1000..` (1000명 이상 푼 문제)
* **`@` 또는 `solved_by:` (특정 유저 해결)**
  * **기능:** 특정 유저가 `OK`를 받은 문제만 필터링
  * **예시:** `@tourist`
* **`tried_by:` (특정 유저 시도)**
  * **기능:** 특정 유저가 제출 기록은 있으나 아직 `OK`를 받지 못한 문제 필터링

### ④ 나의 상태(Status) 검색
* **`status:solved` 또는 `s@me`**
  * **기능:** 내가 시도하여 'OK'를 받은(맞은) 문제만 출력
* **`status:unsolved` 또는 `~s@me`**
  * **기능:** 내가 한 번도 시도하지 않았거나, 아직 풀지 못한 문제만 출력
* **`status:tried` 또는 `t@me`**
  * **기능:** 내가 제출 기록은 있으나 아직 'OK'를 받지 못한 문제만 출력

### ⑤ 대회(Contest) 검색
* **`contest:` 또는 `c:`**
  * **기능:** 특정 대회 번호에 출제된 문제들을 필터링
  * **예시:** `c:1520` (1520번 대회의 모든 문제), `c:1500..1600` (1500번~1600번 대회 구간 검색)

## 2. 논리 연산자 및 결합 문법 (Logical Operators)

각각의 독립된 검색 조건들을 조합하여 더욱 정교하고 복잡한 필터링을 수행할 수 있습니다.

* **공백 (AND / 교집합)**
  * **기능:** 기재된 모든 조건을 동시에 만족하는 문제만 출력
  * **예시:** `#dp *g1` (DP 문제 **이면서** 골드 1 난이도인 문제)
* **`|` (OR / 합집합)**
  * **기능:** 기재된 조건 중 하나라도 만족하는 문제 출력
  * **예시:** `#math | #greedy` (수학 **이거나** 그리디 문제)
* **`~` 또는 `!` (NOT / 차집합)**
  * **기능:** 해당 조건을 만족하는 문제를 검색 목록에서 제외
  * **예시:** `~@me` (내가 푼 문제 제외)
* **`()` (괄호 / 우선순위 지정)**
  * **기능:** 연산의 우선순위를 묶어 복잡한 논리식 구성
  * **예시:** `~(#math | #geometry) *s5..g5` (수학과 기하학을 **모두 제외**하고, 난이도가 실버5~골드5인 문제)
