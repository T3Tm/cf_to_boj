# 🐛 Bug Fix Report (v3.3.9)

## 1. 제출 버튼 동작 불능 해결 (Form Context Integrity)
- **증상:** '제출' 버튼을 눌러도 아무런 반응이 없거나 코드포스 서버에서 거부됨.
- **원인:** 입력 요소와 버튼을 원본 `<form>` 태그 외부로 이동시킴으로써 폼 컨텍스트가 깨짐. 또한 CSRF 토큰 등 숨겨진 필드들이 전송되지 않음.
- **해결:** 원본 `<form>` 태그를 그대로 유지한 채 내부의 시각적 요소들만 재배치함. 모든 입력창과 버튼이 `<form>`의 자식 요소로 유지되도록 설계하여 제출 로직의 무결성을 확보함.

## 2. 레이아웃 잔재 제거 및 초기화 (Visual Cleanup)
- **증상:** "Submit solution", "Codeforces Beta Round..." 등 기존 텍스트가 백준 스타일 폼 위에 노출됨.
- **해결:** `#pageContent` 내부의 모든 요소를 비운 뒤, 탭 메뉴와 원본 폼(재구성된 상태)만 다시 삽입하는 방식으로 페이지를 완전히 초기화함.

## 3. 소스 코드 입력창 가시성 확보
- **증상:** Ace Editor로 인해 원본 `textarea`가 숨겨져 있어 제출할 코드가 보이지 않음.
- **해결:** `textarea[name="source"]`의 `display: none`과 `hidden` 속성을 강제로 제거하고, Ace Editor 관련 `div`를 삭제하여 순수 텍스트 입력창만 노출함.
