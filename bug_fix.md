# 🐛 Bug Fix Report (v3.3.4)

## 1. 제출 페이지 폼 중복 현상 (Form Duplication)
- **증상:** 기존 코드포스의 제출 폼과 새로 생성한 백준 스타일 폼이 한 페이지에 동시에 나타남.
- **원인:** `.roundbox` 요소를 타겟팅하여 교체하는 로직이 코드포스의 복잡한 레이아웃 구조에서 특정 상황에 실패하거나, 다른 컨테이너가 남아있음.
- **해결:** `init` 시점에서 기존 폼의 부모 컨테이너를 더 상위 레벨에서 완전히 제거하거나 `display: none` 처리를 강화하고, 새 폼만 단독으로 렌더링하도록 수정.

## 2. 제출 페이지 데이터 표시 형식 미흡
- **증상:** '문제' 라벨에 문제 번호만 표시되고 제목이 누락됨.
- **원인:** 초기 설계 시 제목 데이터를 가져오는 로직이 부재했음.
- **해결:** `Fetcher`를 통해 문제 제목을 가져와 '4A - Watermelon' 형식으로 표시하도록 개선.

## 3. 탭 메뉴 및 CSS 레이아웃 불일치
- **증상:** 탭 메뉴와 본문 폼 사이의 간격 및 정렬이 백준과 상이함.
- **원인:** CSS 명시도 부족 및 HTML 구조 차이.
- **해결:** `overrides.css`와 `submit.js`의 인라인 스타일을 동기화하여 백준의 1단 레이아웃을 정밀 재현.
