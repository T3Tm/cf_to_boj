# 🏆 Codeforces to Baekjoon Master Specification (v3.16)

본 문서는 모든 버그 수정 및 방어 로직이 통합된 v3.16 최종 아키텍처 명세서입니다.

---

## 🛠️ 주요 업데이트 사항 (Final Patch)

### 1. 렌더링 무결성 (Rendering Integrity)
- **원본 테이블 영구 폐쇄:** 페이지 로드 즉시 `.datatable`과 `.pagination`을 `display: none` 처리하여 표시 개수 변경 시 원본이 튀어나오는 현상을 원천 차단.
- **Fail-Safe API:** API 서버 다운 시 하얀 화면 대신 에러 안내 카드를 출력하는 예외 처리 도입.

### 2. 이벤트 생명주기 (Event Lifecycle)
- **이벤트 위임(Delegation):** 가상 테이블 버튼과 알약 삭제 버튼에 이벤트 위임 패턴을 적용하여, 재렌더링 시 이벤트가 유실되거나 메모리가 누수되는 문제 해결.
- **마운트 동기화:** 헤더와 검색창이 DOM에 완전히 삽입된 후 요소를 쿼리하도록 실행 순서 강제.

### 3. 지능형 쿼리 엔진 (Smart Query Engine)
- **다중 논리 회로:** `|` (OR)와 `공백` (AND) 연산자를 계층적으로 처리하여 복합 검색 지원.
- **수학적 방어:** 범위 검색(`..`) 시 숫자가 아닌 값이 들어와도 `NaN` 에러 없이 무시하거나 기본값으로 처리하는 정규화 로직 탑재.
- **언더바 치환:** `tag:brute_force` 입력 시 자동으로 `brute force`로 해석하는 편의 기능 유지.

### 4. UI/UX 리마스터 (Global Theme)
- **헤더 재건축:** 낡은 코드포스 헤더를 파괴하고 Flexbox 기반의 모던 백준 GNB 주입.
- **동적 페이지네이션:** 20, 50, 100개씩 보기 토글 기능 및 페이지 이동 스크롤 초기화 적용.
- **데이터 결합:** 문제 리스트에 API 기반 '맞힌 사람 수' 데이터 동기화 완료.
