# 🏆 Codeforces to Baekjoon 문제 페이지 퍼펙트 클론 (v3.23)

본 업데이트는 코드포스의 문제 풀이 페이지를 백준(BOJ) 오리지널 디자인과 100% 동일한 구조로 변환하는 프론트엔드 DOM/CSS 리마스터입니다.

---

## 🛠️ 주요 업데이트 사항

### 1. 상단 정보 테이블(Info Table) 재구축
- **변경 사항:** 코드포스 특유의 줄글 형태 제약 조건(Time limit, Memory limit 등)을 파싱하여, 백준 고유의 **중앙 정렬된 회색 테이블(`<table>`)** 형태로 완전히 재조립했습니다.

### 2. 레이아웃 및 번역 자동화 (i18n & Layout)
- **섹션 타이틀:** 본문의 'Input', 'Output', 'Note' 등을 자바스크립트가 감지하여 자동으로 '입력', '출력', '힌트'로 번역하고 굵은 밑줄을 추가합니다.
- **예제 2분할 레이아웃:** CSS Grid(`grid-template-columns: 1fr 1fr`)를 도입하여, 백준처럼 **예제 입력과 예제 출력이 좌우로 나란히** 배치되도록 수정했습니다.
- **예제 번호링:** 'Input'으로만 적혀있던 것을 '예제 입력 1', '예제 출력 1' 형태로 자동 넘버링합니다.

### 3. 알고리즘 분류 (태그) 복구
- **변경 사항:** 전역 사이드바 철거 시 함께 숨겨졌던 태그 데이터(`.tag-box`)를 자바스크립트로 탈취하여, 문제 본문 맨 밑에 백준의 **'알고리즘 분류' 뱃지 형태**로 렌더링합니다.

### 4. 탭 메뉴 유지
- 이전 버전에서 복구했던 '문제', '제출', '채점 현황' 탭 메뉴는 상단에 깔끔하게 유지됩니다.
